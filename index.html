About

Matin (French for "morning") is a ecosystem to develop scalable and sustainable innovative technologies for the Materials Genome Initiative. Currently, materials science information exists in a lawless wild west; it is unstructured, it is searching for comrades, it is fighting for survival.

Workflows

Workflows create and transform information. Workflows can range from fully manual to fully automated.

  1. Sensor Workflows - Sensor workflows create raw data. All the provenance of a information originates from a sensor workflows. A sensor workflow performs a task to generate raw data
  2. Structuring Workflows - A cornerstone of Matin program is our dedicated focus to the user. Structuring workflows identify unstructured sensor data and export structured metadata and data.
  3. Filtering Workflows - Filtering workflows modify information contained within a SINGLE dataset.
  4. Combo Workflows - Combo workflows ingest existing information that can be compared in a deterministic frame. Combo workflows merge heterogeneous streams of information MULTIPLE datasets in a combined dataset. A combo workflow is identified when the metadata for either data source is modified. An example, two sources including (1) an image of a volume and (2) point data sampled within the volume are combined using a Combo workflow; alone the position in (2) are relative while after the combo workflow the position are now absolute to a frame.
  5. Analytics Workflows - Analytics workflows ingest existing information from MULTIPLE datasets that can be compared in a stochastic frame. These workflows will often ingest a lot of data to create compact knowledge of the relationship between the inputs.
  6. Content Workflows - Content workflows strictly operate on metadata. These workflows will dynamically enhance the project metadata by improving metadata description and knowledge dissemination using both visualization and text. All Content Workflow changes are recorded by a versioning system requiring that no modification is ever permanent.
